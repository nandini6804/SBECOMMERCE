E-Commerce CRUD Application
 Project Overview

This is a Spring Boot based E-Commerce CRUD application that allows users to manage products efficiently.
The application performs basic operations like Create, Read, Update, and Delete (CRUD) using REST APIs.

 Features: 
 Add new products
 View all products
Update product details
 Delete products
 RESTful API integration
 Database connectivity using MySQL

 
 Technologies Used:
Java
Spring Boot
Spring Data JPA
Hibernate
MySQL
Maven
REST API
HTML, CSS, JavaScript (Frontend basics)
